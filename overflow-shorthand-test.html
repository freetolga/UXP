Expected Behavior

  • Single values: Should apply the same value to both overflow-x and overflow-y
  • Two values: First value applies to overflow-x, second to overflow-y
  • visible: Content should be visible outside the container
  • hidden: Content should be clipped at container boundaries
  • scroll: Scrollbars should always be visible
  • auto: Scrollbars should appear only when needed
